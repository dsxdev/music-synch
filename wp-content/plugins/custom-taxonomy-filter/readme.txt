This plugin adds custom taxonomy filter in wordpress admin post listing panel as per setting

== Description ==

Custom Taxonomy Filter in WordPress Admin Panel Post Listing Plugin will solve or help to add custom taxonomy filter/dropdown for built-in or custom post type listing in Admin panel or dashboard.

Features:

*   Supports all post types including built-in and custom post types

*   Enable/Disable for any post type
*   Enable/Disable any taxonomy

== Installation ==

How to install the plugin and get it working.


1. wordPress Codex has a very good instruction for multiple ways to install a plugin Upload `customtaxfilterinadmin` folder  to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. From 'Setting' menu see sub menu 'CBX Custom Tax Filter', save setting
4. Go to any post listing from admin panel and check the new dropdown list for the custom taxonomy, also it works for existing taxonomy
